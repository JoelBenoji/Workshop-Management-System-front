# mecX (frontend)


## Description<br>
Front end code repository for a Workshop Management System, which can be used for booking services and to view services, from the Customer and Employee perspective

<b>Technologies Used:</b> ReactJS, Framer Motion

<b>Backend code:</b> https://github.com/JoelBenoji/Workshop-Management-System-back
